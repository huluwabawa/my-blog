# java 中好用的库
## 好用的表达式引擎
~~~ text 
Aviator是一个高性能、轻量级的java语言实现的表达式求值引擎，主要用于各种表达式的动态求值。
~~~
引入`jar包`
~~~ xml
<dependency>
  <groupId>com.googlecode.aviator</groupId>
  <artifactId>aviator</artifactId>
  <version>3.0.1</version>
</dependency>
~~~
